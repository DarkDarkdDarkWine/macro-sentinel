# Macro Sentinel

宏观哨兵 — 全球事件追踪 + 金融市场监控 + AI 驱动的投资建议系统。

## 项目目标

在全球局势不稳定的背景下，构建一套个人使用的宏观投资辅助工具，帮助：

1. **追踪全球大事件** — 实时感知地缘政治、经济政策、黑天鹅事件
2. **监控金融市场波动** — 跟踪 VIX、主要市场指数、汇率、商品价格
3. **推荐投资领域** — 基于宏观信号判断当前应关注哪些行业或资产类别
4. **持仓建议** — 输出结构化的仓位调整建议（非实盘交易）

## 架构设计

系统分三层：

```
┌─────────────────────────────────────────────────────┐
│  第一层：信息采集                                      │
│  - 全球新闻：RSS（BBC、AP、CNBC、SCMP、新华社、NHK 等）           │
│  - 市场数据：yfinance（VIX、指数、汇率、商品）           │
└────────────────────────┬────────────────────────────┘
                         │
┌────────────────────────▼────────────────────────────┐
│  第二层：AI 分析引擎                                   │
│  - 新闻摘要与风险评级（LLM）                           │
│  - 市场情绪判断                                       │
│  - 行业轮动信号                                       │
└────────────────────────┬────────────────────────────┘
                         │
┌────────────────────────▼────────────────────────────┐
│  第三层：投资建议输出                                   │
│  - 宏观风险评级（低 / 中 / 高 / 极高）                  │
│  - 推荐关注的资产类别与行业                             │
│  - 仓位结构建议（防御 / 均衡 / 进攻）                   │
└─────────────────────────────────────────────────────┘
```

## 技术选型

| 模块 | 技术 |
|------|------|
| LLM | DeepSeek API |
| 市场数据 | yfinance |
| 宏观数据 | FRED API |
| 新闻数据 | GDELT |
| Web 框架 | FastAPI + uvicorn |
| 运行环境 | Python 3.10+ |
| 部署 | 本地 / NAS Docker |

## 快速启动

```bash
pip install -r requirements.txt
cp .env.example .env   # 填入 FRED_API_KEY
python3 -m uvicorn src.api.server:app --host 0.0.0.0 --port 8765 --reload
```

打开 http://localhost:8765，点击「采集数据」即可查看实时数据面板。

## 定位说明

本项目为**个人研究工具**，不提供实盘交易功能，输出结果仅供参考，不构成投资建议。
