# 数据源规划

## 第一版接入（v1）

| 数据源 | 类型 | 需要注册 | 用途 |
|--------|------|---------|------|
| **yfinance** | 市场数据 | ❌ | VIX、主要指数、汇率、商品（黄金/原油） |
| **FRED** | 宏观指标 | ✅ 免费 | 利率、通胀、GDP、就业数据 |
| **GDELT** | 全球新闻事件 | ❌ | 地缘政治风险、全球事件追踪 |

> FRED API Key 申请：fred.stlouisfed.org/docs/api/api_key.html

## 后续扩展（v2+）

| 数据源 | 类型 | 需要注册 | 用途 |
|--------|------|---------|------|
| **NewsAPI** | 新闻聚合 | ✅ 免费 | 英文主流媒体覆盖，补充 GDELT |
| **AkShare** | A股 / 宏观 | ❌ | A股行情、国内宏观数据 |

## 东西方信源配对原则

接入新闻类数据源时，必须保证东西方视角平衡：

| 西方视角 | 东方视角 |
|---------|---------|
| Reuters / AP（GDELT 覆盖） | 新华社 / CGTN RSS |
| NewsAPI 英文媒体 | 待定 |
